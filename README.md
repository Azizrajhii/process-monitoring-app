# Plateforme MERN - PFE

Architecture avec deux dossiers indépendants seulement :

- [client](client) : frontend React + Vite + TypeScript
- [back](back) : backend Node.js + Express + MongoDB

## Lancer le frontend

Dans [client](client) :

```bash
npm install
npm run dev
```

## Lancer le backend

Dans [back](back) :

```bash
npm install
npm run dev
```

Copier [back/.env.example](back/.env.example) vers `.env` avant de lancer le backend.

## Ports par défaut

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:5000`
