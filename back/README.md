# Back

Backend Node.js + Express + MongoDB.

## Dossiers

- `src/config` : configuration base de données
- `src/controllers` : logique HTTP
- `src/models` : schémas Mongoose
- `src/routes` : endpoints API
- `src/middlewares` : gestion erreurs et routes absentes
- `src/services` : logique métier réutilisable
